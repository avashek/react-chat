A Chat App using reactjs and socket.io  

### Steps :
##### 1. In the project directory :  
`npm install`  
##### 2. Client setup :  
`npm start`  
##### 3. Server setup :  
`cd chat-server`  
`node index.js`  

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).
